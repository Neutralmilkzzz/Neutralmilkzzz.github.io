这一章足够我们搞定90%的东西了，剩下的10%也没必要现在学。
`git fetch`就是拷贝远程仓库，但是对现在现有的仓库没有任何影响，单纯是抄过来。
`git pull` = `git fetch` + `git merge`
`git pull --rebase` = `git fetch` + `git rebase`
`git push`就是传过去。
差不多了 收工。
